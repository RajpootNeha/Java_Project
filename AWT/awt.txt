Java AWT Programming

AWT:abstract window Toolkit

API to develop GUI or window based application

AWT component are platform dependent
java.awt package is used

Important Methods of Component class
1)public void add(Component c)//inserts Component
2)public void setSize(int width,int height)//Set the size
3)public void setLayout(LayoutManager m)//defines layout
4)public void setVisible(boolean status)//changes the visibility

Java AWT Simple Example[Extend Frame]

import java.awt.*;
class awt_test extends frame
{
	awt_test()
	{
	Button b=new Button("click here");//label of button
	add(b);//add button on fram
	b.setBounds(30,100,80,30)//set position of button
	setSize(400,400);//set frame size
	setLayout(null);
	setVisible(true);
	}
	public static void main(String args)
	{
	awt_test a1=new awt_test();
	}
}






AWT UI Elements
1)Label:Does not create any event
	-object of Label(Passive).
	-Text cannot be changed be end user.
	Label l=new Label("EEC");
	l.setText("CSE");
	l.getText()//retrun string value.
2)TextField:Allows the user to edit single line of text.
	-event is generated by user typing in textfield.
	TextField tf=new TextField(10);
	tf.setText("");

3)Button:Geneartes an event when pressed/released.
	Button b=new Button("click here");

4)CheckBox:Used to turn on option true or false.
	Java doesn't have any radio class in AWT
Checkbox c1=new Checkbox("cse",null,true)//true means by default checked
Checkbox c2=new Checkbox("It",null,false)
CheckboxGroup cbg=new CheckboxGroup();//radio
5)List:Represents a list of text items
	-user can choose either one or multiple items.
	List l=new List(2,true);//true means multiple select are allowed
	l.add("CSE");
	l.add("IT");
	
6)Choice:Used to show popup menu of choices.
	-selected choice is shown on the top of menu
	Choice c=new Choice();
	c.add("CSE");
	c.add("IT");
	
7)TextArea:TextArea ta=new TextArea(5,50);
8)Menu:  a)MenuBar (b)MenuItem
	
9)ScroolBar	: a)Horizontal b)vertical

ScrollBar sb1=new ScrollBar();
sb1.setOrientation(ScrollBar.Horizontal);
sb1.setOrientation(ScrollBar.Vertical);
